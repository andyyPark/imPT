# Overview

FPFS is an open-source software for fast and accurate shear estimation.

## Basic installation

For stable version:
```shell
pip install impt
```

Or clone the repository:
```shell
git clone https://github.com/mr-superonion/impt.git
cd impt
pip install -e .
```
